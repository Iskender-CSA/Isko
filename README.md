1. Название проекта
   Проверка email (Simple Version)

2. Описание
   Этот проект написан на языке Python.
   Программа проверяет, есть ли в email символы "@" и ".".

3. Как работает программа
   Пользователь вводит email.
   Программа проверяет, есть ли в тексте символы "@" и ".".
   Если оба символа есть — email считается правильным.

4. Пример работы

Введите email: [test@gmail.com](mailto:test@gmail.com)
Email выглядит правильно

Введите email: testgmailcom
Email неправильный

5. Логика проверки

* В email должен быть символ "@"
* В email должна быть точка "."
